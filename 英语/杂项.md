# there is 与 is there

```
一、Is there 和There is 的区别如下：

1、使用句型不同：Is there用在疑问句中，回答有肯定或否定两种；There is用在陈述句中。

2、使用形式不同：Is there用在开头，是一般疑问句形式；There is是肯定句形式，否定形式是there isn't。

3、中文含义不同：Is there意思是“那里有什么事情；或那里有什么东西。”There is意思是“那里有什么事情呢？哪里有什么东西呢？”

4、句中关于“一些”的表达形式不同：Is there中要用any，There is要用some。

二、Is there 和There is 的不同用法如下，以例句说明：

例句1：Is there anything wrong with your ears? Yes, there is./ No, there isn't.（翻译：是你的耳朵有毛病吗？是的，耳朵有毛病。/不是，耳朵没毛病。）

例句2：There is no endless winter in the world.（翻译：世界上没有无尽的寒冬。）
```
